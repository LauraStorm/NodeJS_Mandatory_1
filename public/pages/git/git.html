Git commands

Git has a lot of tools / commands for version of control and collobaration.

3 important commands

1. git add

 $ git add filename.js

 // Use git add when you need to 
    add a file to your git repo. You can also 
    use git add -A

2. git commit -m 

 $ git commit -m "write a message"

 // when you commit, remember to write a 
    message that make sense for your commit

3. git push

 $ git push

 // sends your commit to your repo

Other commands

git clone

 $ git clone github-link // remember to stand in the right folder when you clone

git pull

 $ git pull // will download the content from the remote repo
