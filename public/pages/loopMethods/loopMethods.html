Loop Methods

There are different ways to loop over a sequence of data, find specific elements, make changes etc.

In modern JS always use loop methods

Loop-methods:

See examples further down - how to use

.forEach()
.filter()
.map()
.find()

forEach()

  • Use forEach () when you want to iterate over 'each' element in an array
  • Is mostly used for the dom
Example

const friends =["Laura", "Karen", "Yvonne", "Søren"];

friends.forEach(friend => console.log(friend));

// output: Laura Karen Yvonne Søren

filter()

  • filter() creates a new array (it do not override the old/original)
  • Use filter() when you want to work with some of the elements from the array.
  • You filter your array with a condition where ever element will be checked - if they passed the condition
    filter returns a boolean (thats how it checks)
  • It is not a 'one 2 one' because we do not take every element from the original array to the new
Example

const friends = [
  { name: "Laura", age: 26 },
  { name: "Karen", age: 55 },
  { name: "Yvonne", age: 10 }
];

const friendsOver18 = friends.filter(friend => friend.age > 18);

// output:[ { name: 'Laura', age: 26 }, { name: 'Karen', age: 55 } ]

map()

  • map() creates a new array with the same length as the original/old array
  • It is a 'one 2 one'
  • Use filter() when you want need to transform the data from one format to another
    Example: from object to strings or adding a property(key-value) to all objects in an array etc.
Example: from objects to array with strings

const friends = [
  { name: "Laura", age: 26 },
  { name: "Karen", age: 55 },
  { name: "Yvonne", age: 10 }
];

const friendsNames = friends.map(friend => friend.name);

// output:[ 'Laura', 'Karen', 'Yvonne' ]


Example: modify a property

const friendsOver18 = friends.map(friend => ({... friend, age: friend.age+1}));

// output:[{ name: 'Laura', age: 27 }, { name: 'Karen', age: 56 },{ name: 'Yvonne', age: 11 }]

Explain: The arrow function takes a friend as parameter.
We use the spread operater (({...})) syntax to create a copy of the original friend object.
Last we change the age property and increment every friend-objects age.

find()

  • Use find() to find a element in an array
  • It will return the first found element in the array
  • If the element is not found it will return undefined
Example

const friends = [
  { name: "Laura", age: 26 },
  { name: "Karen", age: 55 },
  { name: "Yvonne", age: 10 }
];

const firstFriend = friends.find(friend => friend.age > 10);

// output:{ name: 'Laura', age: 26 }

findIndex()

  • Use findIndex() when you would like to find the index position of an element in an array
  • It will return the first found element in the array taht pass the test/condition
  • If the element is not found it will return -1
Example

const friends = [
  { name: "Laura", age: 26 },
  { name: "Karen", age: 55 },
  { name: "Yvonne", age: 10 }
];

const firstFriend = friends.findIndex(friend => friend.age > 10);

// output: 0

NOTE

For-loops are only used when finger counting is needed

Why you shouldn't use for-loops:

  • Loop methods are shorter (brevity)
  • clarity
  • No accidental errors
