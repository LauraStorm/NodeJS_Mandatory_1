Strings

  • Strings = 'text' in normal language
  • You know you are working with Strings if you se qoutation marks ""
3 ways to write Strings:
// single quotes

const teamOne = 'Team 1';

// double quotes

const teamOne = "Team 2";

// string template literal

const teamThree = `Team ${1+2}`;

String litereal
  • String template literal is a smart way to do String interpolation
  • As shown above the syntax is: ` ${} `
  • Inside ${} you can write variable names etc.

Undefined

  • Undefined indicates that a variable has not been assigned
Example
// the declared variable x has no value assigned, and will be undefined.

let x;

// output: undefined

console.log(typeof (x));

Object

  • Objects can contain many values
  • Values are written in key-value pairs. - (a property).
  • Key = is always a String (think of it like a variable name/the property)
  • Value = is the value to store (like the value to a variable)
Example

const person = {
  firstName: "Laura",
  lastName: "Storm"
};

// key = firstName
// value = lastName
Nice 2 know

The following examples will refer to the person object above

1. Access a property

Here are 2 examples:

const firstName = person.firstName;
// person = the object,
// firstname = key

console.log(firstName);
// output: Laura

const firstName = person['firstName'];

console.log(firstName);
// output: Laura

2. Add a property

Example:

person.age = 26;

// we have now added age to the person object

console.log(person)

// output: { firstName: 'Laura', lastName: 'Storm', age: 26 }

3. Remove a property

To remove a property- use the delete keyword

Example:

delete person.age;

console.log(person);

// output: { firstName: 'Laura', lastName: 'Storm'}

Array

  • Array are of type objects
  • You can tell you are working with arrays by '[]'
  • Arrays are simular to objects --> they both collections of values
    Objects use 'string-keys', Arrays use numeric indices to access and store values
Example:

const friends = [];

// The example is an empty array of friends


Nice 2 know

Working with arrays you might use methods like:

1. Add element to array

Example with: .push()
Using .push() you add the element to the end of the array

const friendOne = 'Søren';

friends.push(friendOne);

friends.push('Karen');

// console.log(friends);

// output: [ 'Søren', 'Karen' ];


Example with: .splice()
With splice you can add multiple elements at the same time and place them in the array from a specific index

const friends = ["Søren", "Karen"];

friends.splice(1,0, "Peter", "Yvonne");

console.log(friends);

// console.log(friends)

// output: [ 'Søren', 'Peter', 'Yvonne', 'Karen' ];

1 = index start - the new elements will be placed from index 1
0 = tells that no elements after should 'disappear' (if it said 1 instead Karen would be removed from the array)
Peter and Yvonne = new elements to the friends array

2. Access elements

Example with: index position
Note: Here we know the index position

const friends = ["Søren", "Peter", "Yvonne", "Karen"];

const firstFriendInArray = friends[0];

console.log(firstFriendInArray);

// output: Søren


Example with: for-loop
Note: for-loop is an 'old' way of accessing elements.
---> use loop methods instead.

const friends = ["Søren", "Peter", "Yvonne", "Karen"];

for (const friend of friends) {
  console.log(friend)
};

// output: Søren Peter Yvonne Karen

3. Remove an element

Example with: .pop()
Using .pop() you will always remove the las element in your array

const friends = ["Søren", "Peter", "Yvonne", "Karen"];

friends.pop();

// removes Karen from the array

// output: Søren Peter Yvonne


Example with: .findIndex() + .splice()
When you would like to remove a element from a specific index position

const friends = ["Søren", "Peter", "Yvonne", "Karen"];

const foundIndex = friends.findIndex(friend => friend === 'Peter');

const deleteFriend = friends.splice(foundIndex, 1);

// Peter is removed from the array

.findIndex = loops through every friend in the array to find 'Peter's index position
.splice = removes Peter from the array. 1 = tells how many elements need to be removed
