Nodemon

  • Nodemon is a tool that is good in production for developers
  • Nodemon will run the script automatically everytime you make a change in your code
  • Use nodemon instead of running $ node app.js everytime you make a change in your code
  • See how to install nodemon here if you dont have it

Expand nodemon

  • You can create a nodemon.json file where you can expand it functionality so it automaticaly update when you make changes in example html-, JS files
  • When the file is created, you can just run the $nodemon app.js and it will work
How to use nodemon:

Instead of writing:

$ node app.js


Write:

$ nodemon app.js

How to expand nodemon:

1. Create nodemon.json file

2. In nodemon.json file write 

{
    "ext": "js,json,html"
}

Postman

  • We use postman for testing our api's
  • Example to test CRUD functionality

Example in postman: GET all birds

2. Write the GET endpoint to get all birds

3. Remember to run your application

Vercel

  • Vercel is a platform we can use to deploy our application
  • You can connect your github repo
  • Remember to create a vercel.json file with configurations before connecting your repo. - find the configuration here or look at 'How to deploy'
  • Log in on Vercels site here

    • OBS: all your javascript files and css files nedd to be moved in to you html page before it works...
How to deploy:

1. Create 'vercel.json' file in your project

2. In vercel.json write following configuration.
// 'app.js' is where the application run. So if it has another name in your project, remember to change it! 

{
    "version": 2,
    "builds": [
        {
        "src": "app.js",
        "use": "@now/node"
        }
    ],
    "routes": [
        {
        "src": "/(.*)",
        "dest": "app.js"
        }
    ]
}

3. Remember to push your project to Github

4. Go to Vercels site and login

5. Select 'Add New..' and choose 'Project'

6. Select the Github repo you would like to deploy

7. Fill out ' Project Name'

8. 'Root Directory' click 'Edit' and choose the project

9. Click 'Deploy
