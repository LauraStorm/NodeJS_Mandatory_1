Date

Date is an object in JS.
There are different methods you can use when you want/need to work with dates and time.

Methods you can use:

See examples further down - how to use

.getDate()
.getDay()
.getHours()
.getFullYear()
new Date()
see more methods
here
Example using different date methods:

1. Get todays date .getDate();

const todaysDate = new Date().getDate();

// will return the date (a number) ex. 14

2. Get day .getDay();

const today= new Date().getDay();

// will return a number of the day in the week (0-6)

Note:
Sunday = 0, Monday = 1 ... Saturday = 6
3. Get hour .getHours();

const today= new Date().getHours();

// will return a number of hours (0-23)

4. Get year .getFullYear();

const today= new Date().getFullYears();

// will return the year (ex. 2023)

5. create a date new Date();

const myBirthday = new Date("february 6, 1997");

const myBithdayDate = myBirthday.getDate();

console.log(myBithdayDate);

//output: 6

Note:
There are other ways you can create/write the date in the ()
When you console.log(myBirthday) the output is 1997-02-05T23:00:00.000Z.. Notice the date is 'wrong'- but it is not (confusing)
