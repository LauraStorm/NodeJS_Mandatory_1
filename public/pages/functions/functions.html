Functions

Function Declaration

console.log(sum(8,2));

// output: 10


function sum (num1, num2) {
  return num1 + num2;
};

Notes:
When you use function declaration you can call/use the function anywhere in your script.
It is possible to use the function before it is declared. - it's because function declaration are hoisting ('to lift up')

Callback Functions

function greet (aName, aFunction) {
  return aFunction(aName);
};


function sayHello (aName) {
  return `Hello ${aName} - have a lovely day!`;
};


console.log(greet("Laura", sayHello));

// 'sayHello' is a callback function because we 'call' the function and execute it in 'greet' function

// output: Hello Laura - have a lovely day!


Notes:
A callback function is a function passed into another function as an argument.
The 'outher function' will use the passed function and 'call' it

Anonymous Functions

Example

const fullname = function (firstname, lastname) {
  return `${firstname} ${lastname}`;
};


console.log(fullname("Laura", "Storm"));

// output: Laura Storm

// as you see there is no function name after the function keyword



Example - arrow function

const fullname = (firstname, lastname) => {
  return `${firstname} ${lastname}`;
};


or

const fullname = (firstname, lastname) => `${firstname} ${lastname}`;


console.log(fullname("Laura", "Storm"));

// output: Laura Storm


Notes:
As the name says, it's a function without a name --> the function is anonymous.
There are different syntax when you write anonymous functions.

Arrow functions ' => ' gives us the opportunity to write shorter functions and more compact.
If the you dont have parameters then you just write empty '()'
