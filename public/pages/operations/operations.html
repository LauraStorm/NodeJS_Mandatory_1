Operations

Equality


When you want to compare we use:
  • Soft equality ' == '
  • Hard equality ' === '
Hard equality

Hard equality checks value and type - and will return a boolean


Example

2 === 2;

// True

2 === "2";

// False - because they are not the same type


Soft equality

Soft equality checks if's equal and do not care about type - and will return a boolean

Example

2 == 2;

// True

2 == "2";

// True - it change the string to a number and compare

2 == 5;

// False
