CommonJS



require()

  • require() is a function that is used to import modules in Node.js
  • It is a part of the CommonJS module system, which is the default module system used in Node.js
  • When we use require() we are specifying a path to a module that you want to import
  • require() = synchronous
    which means that it blocks the execution of the code until the module has been loaded
app.js

// import with require()

const express = require("express");


export

  • tanks.j - exporting
  • app.js - importing
tanks.js 

function getTanks () {
return tanks;
}

function addTank (tank){
tanks.push(tank)
return tanks;
}

module.exports = {
getTanks,
addTank
}


app.js

const {getTanks, addTank} = require("./util/tanks.js");

or

const tanks = require("./util/tanks.js");

// access: tanks.getTanks()


Note

The module will be importet as a object.
Key-value - has the same name in this case (the function names)
EX.
Key = getTanks
Value = getTanks


ECMAScript (ES6)



import

  • import is a keyword that is used to import modules in ECMAScript (ES) modules
  • It is a newer module system that was introduced in ES6.
  • With import we are allowed to specify (destructering) what we would like to import from a module
    (example: import 2 functions out of 4 in module)
app.js

// import with the keyword import

import express from "express";

Import JS file in HTML


How to import a script to html

1. Create a script - example frontpage.js


2. Write a log in that script - so we can check the connection later


3. Create your HTML page - example frontpage.html

Note: write '!' + 'enter' - then you get the 'html start syntax'


frontpage.js

console.log("Test - frontpage");

4. Write a script-tag in frontpage.html
<script src="/pages/frontpage/frontpage.js" type="module"></script>

// src = the path to where your frontpage.js script is placed

// type="module" = tells the browser to 'treat' the script as a module
- we do it when we use ES6 module - import/export

Note: it is possible to do it without the attribute ' type="module" ' but it will handle it as traditional JavaScript and not access the module functionality.
Another thing is to place the script at the bottom of the body-tag. Because the HTML file will be read from top to bottom


5. Run your app.js with: $ nodemon app.js


6. Go to your browser on your frontpage and "undersøg" and choose "console"


7. Your message should be shown

frontpage.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>frontpage</title>
</head>
<body>

    <!-- frontpage.js importet -->
    <script src="/pages/frontpage/frontpage.js" type="module"></script>
</body>
</html>

Import CSS file in HTML


How to import a CSS file to html

1. Create a CSS file - example main.css


2. In your CSS there should be a styling - so you can check that it is connected.
Example: change the bacground color


3. Go to an HTML page where you would like the styling - example frontpage.html


main.css

body {
    background-color: orange;
}

4. Write a link-tag in the head-tag in frontpage.html
<link rel="stylesheet" href="/assets/css/main.css">

// rel = specifies/describe the type we link

// href = the path to your CSS file


5. Run your app.js with: $ nodemon app.js


6. Go to your browser on your frontpage


7. Your background should be orange (in this case)


frontpage.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- main.css importet -->
    <link rel="stylesheet" href="/assets/css/main.css">
    <title>frontpage</title>
</head>
<body>

    <!-- frontpage.js importet -->
    <script src="/pages/frontpage/frontpage.js" type="module"></script>
</body>
</html>
