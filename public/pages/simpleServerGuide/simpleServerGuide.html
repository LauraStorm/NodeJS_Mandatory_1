Setup A Simple Server with express

  • Notice that this guide will use commonJS to import

Steps

Step 1: Create project

  • Create a project in Visual Studio e.g.
  • Create a new file: package.json
package.json

Step 2: Write dependencies

  • In package.json write Express dependency:
Dependencies in package.json 
{
    "dependencies":{
        "express": "4.18.2"
    }  
}

Step 3: Install dependencies

  • Install dependencies: Write $ npm install in the terminal
Install dependencies

$ npm install

Step 4: Create app.js

  • Create a new js file: app.js
    - it's in this file we will setup the server
app.js

Step 5: Setup the server in app.js

  • Declare a variable that 'gets' express module/libary: const express = require('express');
  • Declare a variable 'app' with the express variable: const app = express();
Get express

const express = require ('express');

Declare 'app' variable

const app = express();

Step 6: Setup the Port in app.js

  • Write the port number on the app variable
    - This app starts a server and listens on port 8080 for connections
Get express
const PORT = 8080;
app.listen(PORT, (error) => {
    if (error) {
        console.log(error);
        return;
    }
    console.log("Server is running on port", PORT);
});

Step 7: Execute app.js file in terminal

  • Be in the folder where app.js is
  • Execute app.js in terminal
    node: because it is with node we would like to execute the file
    app.js: is the file we would like to execute
Execute app.js in terminal
$ node app.js

Step 8: Check if server is connected

  • Go to a browser (Google Chrome e.g.)
  • Write localhost:8080
  • The browser should give you this message: cannot GET /
    if it's connected
In a browser
localhost:8080
You are connected if you get this message in browser
cannot GET /

... That was it!
