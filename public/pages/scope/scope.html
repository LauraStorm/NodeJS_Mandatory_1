Scope

Scope is a concept about where variables will be exist and where they no longer will exist.
- when you can access the variable - within and outside a scope.

Global scope vs Local Scope


Global Scope
  • Global scope is when you can access variables and functions from everwhere in the code
Local Scope
  • Local scope is where variables and functions are declared within a function or block.
  • These variables and functions can only be accessed inside the function or bloc where they are defined
